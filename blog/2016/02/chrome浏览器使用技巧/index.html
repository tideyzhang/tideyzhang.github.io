Chrome浏览器使用技巧

张潮 / 2016-02-28

使用 Chrome 很久，还不时发现一些小的技巧，加上许多已经成为习惯的技巧，便在这里记下来，做一个简单的推荐，也方便自己记忆。

1. 历史

Chrome 浏览器是 Google 在08年上线的网页浏览器。Chrome 原意是化学元素「铬」，而「铬」的原意在拉丁文中是「颜色」，再加上 Chrome 在网页设计中有「框架」之意，因此就像 Google 或者 Apple 这些公司在设计自己名字时机缘巧合所产生的各种故事一样，Chrome 也给人以多种遐想。
按照更新频率和稳定性的不同，有四个版本分支(参照 Wikipedia )：

  1. Stable Channel（稳定版／正式版），为追求稳定的普通用户使用。
  2. Beta Channel（测试版），正式版发布前的公开测试所用，版本上的新功能不会作太多修改，主要为安全上的测试，可能会存在不稳定情况，适用喜欢较稳定又可尝鲜的朋友下载使用。
  3. Dev Channel（开发版），主要为功能上的测试，可能存在稳定性问题，更新速度通常为一周一次，适合开发人员使用。
  4. Canary Build（构造版），2010年7月加入，比开发版更新更不稳定，但相对的也会加入更多测试性的新功能，仅适合开发人员或追求紧贴最新版本的用户使用。Canary Build 有一个特色是它可以独立安装及免安装，不会覆盖原本的“ Google Chrome ”版本，让系统中可以同时拥有两套安装版“ Google Chrome ”，但 Canary build 的用户文件不能与其他分支共用。

建议参照自己的折腾能力使用 Stable 版本或者 Beta 版本。至于更新，最简便的方式是翻墙后自动更新。（经测试，软件本体不翻墙可直接在官网下载，扩展程序商店还需要翻墙。）

2. 界面

进入 Chrome 后可以发现其界面简洁而有特点：

最上面是多标签页，可以通过鼠标拖拽移动或者分离。

再往下是融合了搜索功能的地址栏，同一层次还有插件图标以及菜单选项。

最后是可隐藏的书签栏（图中已隐藏）。

三行之后，全部是网页正文，但仍然会在鼠标指向链接时在左下角出现链接地址。
接下来是菜单，直接修改相应参数或查找信息，都是即时操作，不需要点击保存按钮，可说是使用习惯与设计思路的一大飞跃。

除此之外，还有扩展程序，历史记录，书签管理器，下载内容等界面，与一般网页显示方式一样，风格统一。

3. 操作

标签页

标签页是个很好玩的东西，而 Chrome 更是把标签页做到了极致，几乎所有的设置都以标签页的形式表现出来，即时存储。
关于功能，鼠标右键即可，个人最常用的有固定标签页，关闭右侧标签页，重新打开关闭的标签页等。都有快捷方式，但考虑到实用频率，鼠标右键足够。

地址栏

不用输入全部域名（如www.baidu.com），只用输入baidu即可，浏览器会自动进行补全。
地址栏同时有搜索功能，在任何时候直接在地址栏中输入关键字即可使用默认搜索引擎进行搜索（默认搜索引擎在设置中设定）。
快捷操作：

在浏览某网页时Ctrl+L选中地址栏，直接输入网址或关键词，Alt+Enter在新标签页打开。

在设置-->搜索中设置默认搜索引擎。除此之外，Chrome还可以记住用户所浏览网页的搜索框，用户可以指定关键字，在地址栏中实现快捷搜索。如，豆瓣图书的搜索，设置关键词为b。
快捷操作：

Ctrl+L选中地址栏，输入b后，按Tab键，地址栏出现用book.douban.com搜索字样，输入关键字搜索即可。

几乎所有的操作都可以在地址栏中输入到达（都是以chrome:开始），特别是实验室功能入口，chrome://flags/，可以停用或启用某些实验室功能。

书签

书签栏在地址栏下面，可以选择显示或隐藏（个人认为当然显示更方便）。书签可以实现同步，并且也可以与手机版 Chrome 浏览器同步。添加书签栏的操作有这样两个便捷方式：

  1. 地址栏右边五角星图标，点击即可，并可以指定书签文件夹。
  2. 鼠标选中网址，直接拖拽到书签栏相应地方，可以更方便的确定书签位置。
    书签管理器现在有符合 Material Design 原则的页面，可以使用扩展程序 Bookmark Manager 实现。

设置

设置中有很多好玩的地方，自定义关键词在地址栏中实现快捷搜索在前面已经说了，除此之外：

  1. 改变网页字体，在设置-->高级设置-->网络内容-->字号里，个人习惯于自定义为微软雅黑，字号适当调大。
  2. 管理密码，在设置-->高级设置-->密码与表单-->管理密码中，点击后可以查看浏览器保存的密码，当然密码并不可见，需要输入电脑系统的密码才能查看。（以后不用记密码啦！）

扩展程序

扩展程序是 Chrome 的最大特色，以至于越来越像一个微型的操作系统，再浏览器上就可以实现各种各样的功能。仅就浏览方面，以下几种是必备：

  1. Adblock Plus，广告拦截工具，界面清爽了不是一点点。支持自定义。
  2. crxMouse Chrome Gestures，鼠标手势，但又不仅仅是鼠标手势，超级拖拽功能也很便捷，自定义很方便。同样，支持 Google 账号同步。
  3. 另外的一系列诸如保存网页，截图，翻译，稍后阅读，购物助手，繁简转换等，都可以在商店中很方便搜到（需要翻墙）。
  4. 你不知不觉添加了那么多扩展程序，因此有必要对其进行管理了，在不同的环境下使用不同的扩展组，推荐 Context ，可以进行分组并快速切换。
